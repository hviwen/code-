# Source Code

Runtime and reusable study code lives here.

## Boundary

Use `src/` for cleaned-up reference implementations that are worth reusing or testing.
Put scratch rewrites and one-off interview drills in `practice/` first.

- `composables/`: Vue composables and related Pinia-backed helpers.
- `stores/`: Pinia stores used by examples and composables.
- `plugins/`: reusable plugin examples.
- `utils/`: standalone JavaScript utilities and async helpers.
- `data-structures/`: linked list, heap, graph, tree implementations.
- `algorithms/`: reusable algorithm implementations and visual demos.
- `examples/`: Vue examples and browser-facing demos.
