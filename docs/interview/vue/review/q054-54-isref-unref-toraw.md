### 📊 技术点评

#### 🎯 核心考点

#### ✅ 正确答案/参考答案
- isRef：判断值是否为 ref
- unref：返回 ref 的 value 或原值（等价于 isRef ? val.value : val）
- toRaw：返回 reactive 代理的原始对象，修改原始对象不会触发响应

#### 💼 实际应用场景
1. 处理第三方库要求原始对象
2. 统一获取 ref 或普通值

#### ⚠️ 技术纠正（如有）
