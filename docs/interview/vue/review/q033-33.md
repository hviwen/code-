### 📊 技术点评

#### 🎯 核心考点
- 指令注册与钩子
- 场景适配（聚焦/权限/懒加载）
- 面试意图：判断指令是否为首选方案而非组件

#### ✅ 正确答案/参考答案
- 全局：`app.directive('focus', { mounted(el){ el.focus() } })`
- 局部：在组件 `directives` 选项或 `<script setup>` 返回对象中注册
- 指令钩子与组件生命周期同名：created/mounted/updated/beforeUnmount/unmounted 等

#### 💼 实际应用场景
1. 自动聚焦、权限隐藏
