**51.** 什么是 `Fragment`（片段）？Vue 3 中有什么变化？

- Fragment 是 Vue 3 中用于支持组件返回多个根节点的特性。它允许组件模板中不需要额外的包裹元素，减少无意义的 DOM 层级。
